*자바
1)자바란? 가전제품이나 PDA와 같은 소형기기에 탑재될 소프트웨어를 만들기 위해 C++언어의
장점을 그대로 가져오고 단점을 보완하고자 만든 언어

2)자바언어의 특징
 a.운영체제에 독집적으로 실행이 가능하기 때문에 이식성이 좋음.
   자바 에뮬레이터인 자바가상머신(JVM)을 통해서 가능한 것.
    JVM이 운영체제가 이해할 수 있도록 자바코드를 변환하여 전달함.

 b.객체지향언어
   C++과 마찬가지로 상속, 캡슐화, 다형성이 잘 적용된 순사한 객체지향언어

 c.자동 메모리 관리
   가비지컬렉터가 자동적으로 메모리를 관리
 
 d.동적로딩 지원
   프로그램 실행 시 모든 클래스가 로딩되지 않고 필요한 시점에 클래스를 로딩하여 사용 가능.
 
 cf)단점: 에뮬레이터를 거쳐 운영체제에 전달되다보니 코드의 속도가 느림 
           요즘엔 바이트코드(컴파일된 자바코드)를 기계어로 바로 변환해주는 JIT컴파일러와 향상된 최적화 기술이
           적용되어서 속도의 격차를 많이 줄였다고 함.
 
*JVM(Java Virtual Machine)
자바를 실행하기 위한 가상 기계
"Write once, run anywhere!"

*JDK 설치
가성머신과 자바클래스 라이브러리(Java API)외에 개발하는데 필요한 프로그램들
그 프로그램들은 bin 폴더에 들어있음
 1)path설정
   OS가 파일의 위치를 파악하는데 사용하는 경로

*JAVA API
http://java.sun.com 에서 다운가능
